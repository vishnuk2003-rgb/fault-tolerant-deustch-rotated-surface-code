# Fault-Tolerant Deutsch Algorithm using Rotated Surface Code

This repository contains the implementation and simulation of a fault-tolerant Deutsch algorithm for the oracle f(x)=0 using the rotated surface code architecture.

## Features
- Rotated surface code logical qubit encoding
- Stabilizer syndrome extraction
- Minimum-weight perfect matching (MWPM) decoding
- Fault-tolerant Deutsch circuit construction
- Logical vs physical error analys
